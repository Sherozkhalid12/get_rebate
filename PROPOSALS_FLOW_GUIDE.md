# Proposals Flow - Complete User Guide

## üéØ How to Access the Proposals Flow

### For Buyers/Sellers:

1. **Navigate to Agent/Loan Officer Profile**
   - Go to Home tab
   - Search for agents or loan officers
   - Tap on any agent/loan officer card to view their profile

2. **Create a Proposal**
   - On the agent/loan officer profile page
   - Tap the **"Create Proposal"** button (primary blue/green button at the top)
   - Fill in the proposal form:
     - Message (optional)
     - Property Address (optional)
     - Property Price (optional)
   - Tap **"Send Proposal"**

3. **View Your Proposals**
   - Go to **Profile** tab (bottom navigation)
   - Scroll to **Settings** section
   - Tap **"My Proposals"**
   - OR use the floating action button in the proposals view

## üì± Complete Service Lifecycle Flow

### 1. **Proposal Status: PENDING**
- **What you see:**
  - Orange status badge: "Pending"
  - No action buttons (waiting for response)
  
- **What happens:**
  - Agent/loan officer receives notification
  - They can accept or reject your proposal

### 2. **Proposal Status: ACCEPTED ‚Üí IN PROGRESS**
- **What you see:**
  - Blue status badge: "In Progress"
  - **"Complete Service"** button appears (primary blue)
  - **"Report Issue"** link appears below
  
- **What you can do:**
  - Tap **"Complete Service"** when service is delivered
  - Tap **"Report Issue"** if there are problems

### 3. **Proposal Status: COMPLETED**
- **What you see:**
  - Green status badge: "Completed"
  - **"Submit Review"** button (amber/gold color)
  
- **What you can do:**
  - Tap **"Submit Review"**
  - Rate 1-5 stars (animated star selection)
  - Write a review
  - Submit to help other users

### 4. **Proposal Status: REJECTED**
- **What you see:**
  - Red status badge: "Rejected"
  - No action buttons (flow ends)

### 5. **Proposal Status: REPORTED**
- **What you see:**
  - Red status badge: "Reported"
  - Report details shown

## üö® Reporting Issues

If service cannot be completed:
1. Tap **"Report Issue"** (when status is IN PROGRESS)
2. Select reason:
   - Service failure
   - Behavioral issues
   - Delays
   - Unmet expectations
3. Provide description
4. Submit report

## ‚≠ê Submitting Reviews

After service completion:

2. Rate 1-5 stars (tap stars to select)
3. Write your review
4. Tap **"Submit"**
5. Review is saved and visible on agent/loan officer profile
